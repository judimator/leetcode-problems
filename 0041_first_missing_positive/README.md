# The task

Given an unsorted integer array `nums`. Return `the smallest positive integer` that is not present in `nums`.

You must implement an algorithm that runs in `O(n)` time and uses `O(1)` auxiliary space.

### Example 1:

**Input:** nums = [1,2,0]  
**Output:** 3

### Example 2:

**Input:** [3,4,-1,1]  
**Output:** 2
